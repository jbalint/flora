Rules on the Web:

  • Rule Interchange Format (RIF). This is a W3C standard for exchanging rules over the Web. It is based in large part on F-logic and HiLog, especially its Basic Logic Dialect (RIF-BLD) and the Framework for Logic Dialects (RIF-FLD).
