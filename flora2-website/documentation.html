Flora-2 Documentation

Currently there are two documents: the main Flora-2 manual and a manual describing the packages that are not part of the core system, but are important additions and are distributed with the system. In addition, there is a tutorial that covers the main aspects of the theory behind Flora-2 as well as introduces the key features of the system.

  • User's Manual for Flora-2 (PDF) - the latest release.
  • Flora-2 Packages (PDF) - Various packages that extend the core system.
  • Flora-2 Tutorial (PPT, text) - This tutorial has two parts: Foundations and Programming.

The LaTeX source files of the Documentation are available from CVS. You need LaTeX and the associated tools in order to create a printable version.
