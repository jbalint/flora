Download FLORA-2

FLORA-2 relies on the XSB inference engine to run. One can install either the latest released version of FLORA-2, Version 0.99 (Water Lily) or the current development version from the SVN repository.

Note: the old CVS repository that was used by FLORA-2 until now has been deprecated (but is still available).

Downloading the latest release of FLORA-2:

Please follow these steps:

  • Download and install the latest release of XSB.
  • Download the latest release of FLORA-2 into a separate directory and follow the installation instructions.
  • Take a look at the release notes for the list of changes, bug fixes, and incompatibilities with previous versions.

Downloading the current development version from SVN:

This version contains the latest features, but might not always be stable. However, this allows you to track the development of FLORA-2 and contribute! You can install the released version and the SVN version on the same system and switch between them at will (but not within the same session of XSB). The development version of FLORA-2 often requires that you use the latest development version of XSB directly from the XSB's CVS repository. (Under Windows, configuring the CVS version of XSB requires Microsoft's Visual Studio.)

Check out the latest version of FLORA-2 from the SVN repository into a separate directory and follow the installation instructions. If you are not a developer, you can download a read-only copy by clicking on the RO button on that page and copying the appropriate SVN command that appears on the right. Please report to us any problems (and fixes!).

Note: the development version downloaded from SVN is not regularly tested under Windows.
