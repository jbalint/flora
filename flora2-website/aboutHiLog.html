HiLog

Some Papers on HiLog

HiLog is a logical formalism that provides higher-order and meta-programming features in a computationally tractable first-order setting.

  • HiLog: A Foundation for Higher-Order Logic Programming . Journal of Logic Programming, 1993.
    This is the main reference to HiLog. An earlier paper appeared in ILPS'89.
  • Sorted HiLog: Sorts in Higher-Order Logic Data Languages . Intl. Conf. on Database Theory, 1995. Lecture Notes in Computer Science 893. This work extends HiLog with certain typing system. Not implemented in FLORA-2.
  • K.A. Ross, "On Negation in HiLog". Journal of Logic Programming 18(1), 1994.
  • Efficient Execution of HiLog in WAM-based Prolog implementations . Intl. Conference on Logic Programming, 1995.
